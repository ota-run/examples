# Windows-first operating notes

- Keep the `.NET` SDK version in `global.json`.
- Keep PowerShell release behavior explicit in `scripts/release.ps1`.
- Keep cross-platform helpers visible in `ota.yaml` task variants instead of hiding them in shell glue.
- Use `ota doctor` before assuming local toolchain state is still correct.
